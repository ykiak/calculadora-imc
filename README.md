1. APRESENTAÇÃO
Calculadora de Índice de Massa Corporal (IMC) desenvolvida com HTML, CSS e JavaScript puro

2. DEMO
https://ykiak.github.io/calculadora-imc/

3. FEATURES
- Cálculo do IMC a partir do peso, altura e sexo
- Classificação automática por faixa de IMC
- Cálculo do peso ideal mínimo e máximo
- Feedback visual dinâmico por cores
- Conteúdo explicativo e tabela de referência

4. TECNOLOGIAS
- HTML5 (estrutura semântica)
- CSS3 (Grid Layout e estilização)
- JS (manipulação do DOM)
